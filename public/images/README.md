# Place your images here

## Recommended Images:

1. **profile.jpg** - Your professional headshot (256x256px recommended)
   - This will be displayed in the About section
   - If not provided, a placeholder with your initials will be shown

2. **og-image.jpg** (optional) - Open Graph image for social media sharing
   - Recommended size: 1200x630px
   - Will be displayed when sharing your site on social media

## Tips:

- Keep images optimized for web (compress them)
- Use JPG for photos, PNG for graphics with transparency
- Consider using [TinyPNG](https://tinypng.com/) to compress images
- Keep file sizes under 500KB for better load times
